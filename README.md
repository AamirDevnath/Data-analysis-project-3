# Using analytics to drive employee incentive and wellness

Business objective:

- Provide a list of healthy and low absenteeism individuals for the "Healthy bonus program" with a budget of R1000.00.
- Calculate a wage increase/annual bonus for non-smokers with a budget of R982 221.00.
- Create a dashboard to summarise results for non-technical stakeholders.

# My approach:
First established a connection to Microsoft SQL Server and designed a databse before the initial data upload. I then wrote SQL joins to unify datasets and applying specific filtering logic to isolate segments like "healthy workers" for targeted analysis. I integrated business logic directly into the database layer by performing compensation calculation analyses and optimizing performance through refined query structures and CASE statements for conditional data mapping.  To move from raw data to insights, I imported the data Power BI from the SQL database using the queries I designed earlier. I then transformed the data further and created an interactive report.
